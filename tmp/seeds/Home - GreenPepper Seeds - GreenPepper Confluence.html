How to write a seed ?

Proposed Seeds template (to debate)
A good way of writing a seed could be :

  1. Label the seed with the seed label
  2. rewrite the name of the seed using H2 title
  3. sentence and introduction about the aims of the seed
  4. for each rule a description of it must be given
  5. an illustration checking the rule is provided (it is a table or a bullet list which is intended to be executed)
