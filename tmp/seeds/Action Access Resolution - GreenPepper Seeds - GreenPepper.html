Action Access Resolution

Import
GreenPepper
com.greenpepper.seeds.action

I want to be able to access a method, a public field, or a property (getter, setter) of my fixture or SUT:

Rule for Action Access Resolution
access can get a value using? can set a value using?
aPublicField yes yes
APublicMethodWithNoParameterThatReturnNothing no no
APublicMethodWithOneParameterThatReturnNothing no yes
APublicMethodWithNoParameterThatReturnAValue yes no
APublicMethodWithOneParameterThatReturnAValue yes yes
aPrivateField no no
APrivateMethodThatReturnNothing no no
APrivateMethodThatReturnAValue no no

Methods resolution is made by their name and their arity, so you can have N methods with the same name but with a different number of parameters.

Rule for Methods resolving
method called with arity has method been called ?
my method that is override with no parameters and two parameters 0 true
my method that is override with no parameters and two parameters 2 true
my method that is override with no parameters and two parameters 3 false
