The GreenPepper Open project

It was called GreenPepper Open by the creator of GreenPepper when the product was commercial. This project hosted every thing that was OpenSourced and that was a contribution of the community.
We kept that name but we now hosts underneath it anything that makes GreenPepper open to the world :

  • Languages Extensions
  • Wikis Plugins
  • IDE/Build/Countinuous Integration tools
