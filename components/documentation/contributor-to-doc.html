How to contribute to documentation ?

Get the code

We use GitHub to host our project and Code. The documentation itself is OpenSourced and will live and evolve with the contribution of many.

Create an account on Github

Open your browser on http://github.com and register.

Go to http://github.com/strator-dev/greenpepper and get the URI to clone the repository. Then checkout the gh-pages branch. It's the branch containing the documentation.

git clone https://github.com/strator-dev/greenpepper.git
cd greenpepper
git checkout -b gh-pages origin/gh-pages

Run it on Linux

I suggest you install apache httpd and expose the documentation locally. Here are an exemple of setup on Ubuntu

sudo apt-get install apache2
sudo ln -s $(pwd)/greenpepper /var/www/html/greenpepper
Open your browser on the URL : http://localhost/greenpepper
